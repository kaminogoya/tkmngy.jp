上ノ郷谷 太一 (かみのごうや たいち)。デザイナー。1979年3月3日大阪府生れ。東京都在住。

事業会社でプロダクト開発、CI / VI / BI、デザインチーム構築・マネジメント、経営メンバーとして組織の運営に携わってきました。

経歴

  1. 2005年4月〜2012年12月 シックス・アパート株式会社
    • コーポレートウェブサイト構築・運用
    • TypePad のモバイル対応
    • Movable Type 3, 4 の日本語ローカライゼーション
    • Movable Type 5 のプロダクトマネージメント、デザイン、フロントエンド実装、VI
  2. 2013年1月〜2013年5月 Orinoco Peatix 株式会社
    • クレジットカード決済 UI のデザイン、フロントエンド実装
  3. 2013年6月〜2015年2月 クックパッド株式会社
    • 国際版クックパッドの立ち上げ、サービスサイトやアプリのデザイン、フロントエンド実装
    • コーポレートロゴのリニューアルデザイン、VI
  4. 2015年3月〜2019年2月 株式会社トレタ
    • CCO (最高クリエイティブ責任者)
    • CDO (最高デザイン責任者) 2016年7月〜
    • 経営、組織全体へのデザインの啓蒙
    • 予約/顧客台帳アプリ Toreta のブロダクトオーナー、デザイン、BI
    • 経営者、プロダクトオーナーのアイデアの具現化
    • 製品開発部門の構築・運用、マネジメント
    • デザイン部門の構築・運用、マネジメント
